This is a 2d fighting game 

**Concept of the game:**
- Select your fighter.
- Selec your map.
- Each game is 60secs.
- Best out of 3.
- Whoever kills the other one first or deals the most damage in the time frame win.
- Each character will have their own unique skills kit.
